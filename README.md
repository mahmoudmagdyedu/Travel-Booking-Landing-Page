## 🎯 Problem

Travel agencies need a modern, visually stunning web presence that showcases destinations, builds trust through testimonials, and converts visitors into customers with clear pricing and easy booking.

## ✨ Features

- **Hero Section** — Full-screen parallax hero with animated search bar
- **Popular Destinations** — Interactive card grid with hover effects & filtering by region
- **Travel Packages** — Pricing cards with feature comparison
- **Testimonials Carousel** — Auto-rotating customer reviews with navigation dots
- **Booking Form** — Multi-step booking form with validation
- **Stats Counter** — Animated counters for key metrics (destinations, travelers, etc.)
- **Gallery** — Lightbox-style image gallery with category tabs
- **Newsletter** — Email subscription section
- **Fully Responsive** — Mobile-first design, works on all screen sizes
- **Smooth Animations** — Scroll-triggered fade-in and slide animations

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/mahmoudmagdyedu/Travel-Booking-Landing-Page.git

# Open in browser
cd Travel-Booking-Landing-Page
open index.html
```

No build tools needed — just open `index.html` in your browser.
